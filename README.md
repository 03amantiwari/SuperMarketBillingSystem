# SuperMarketBillingSystem
console-based Supermarket Billing System developed in C++ using Object-Oriented Programming (OOP) concepts and File Handling (File I/O).

1. Bill Class
    The backbone of the system, encapsulating all data related to a single customer transaction.

    Attributes: Handles item details, quantities, prices, and calculated totals.

Encapsulation: Uses private members with public getters/setters to ensure data integrity.

2. addBill() Global Utility
    A specialized function designed to interface with the Bill class.

    Functionality: Collects user input for multiple items.
